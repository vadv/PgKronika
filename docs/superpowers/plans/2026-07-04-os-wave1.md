# OS Wave 1 (procfs core) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: use superpowers:subagent-driven-development or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Collect the six host-wide procfs "core" metrics (CPU, stat-misc, meminfo, loadavg, vmstat, PSI) into sealed segments as registry sections `1_102`ŌĆō`1_107`, host-independent and testable from fixture `/proc` trees.

**Architecture:** `kronika-source-os` gains a `ProcFs` reader (root overridable via `KRONIKA_PROC_ROOT`) and one pure parser module per procfs file. Each parser is `fn(content: &str) -> Result<Vec<Row>>` ŌĆö unit-tested on sample strings, no live host. The registry gains one `Section` codec per type. The collector reads all six on one paced `SourceKind::OsCore` tick through a new non-PG `collect_os_sources` path, and BDD drives them end-to-end through a fixture `/proc` root.

**Tech Stack:** Rust (edition 2024, MSRV 1.96), `#[derive(Section)]` macro, Arrow/Parquet codec, cucumber BDD, tokio collector.

## Global Constraints

- MSRV `1.96`; gate every task with `cargo fmt --all -- --check`, `cargo clippy --workspace --all-targets -- -D warnings`, `cargo test --workspace`, `cargo run -p xtask -- check-deps`. Run with `export PATH="$HOME/.cargo/bin:$PATH"` first.
- No new crate dependencies without reason; `kronika-source-os` may add only what parsing needs (it already has `rustix`).
- Comments in English; commit messages in Russian; no `Co-Authored-By`.
- Parsers must not panic on malformed input ŌĆö return `Result`, use checked/saturating conversions.
- `snapshot_full` sections: never write a partial snapshot as full. On read/parse error, skip the section and emit a `collection_degraded` structured log; do not fabricate zeros.
- Every OS row carries a `scope: u8` column (`0=host, 1=pod, 2=pod_net, 3=container, 4=unknown`). Wave 1 procfs-core files leak host-wide inside containers, so their scope is `host` on bare metal and `host` under a detected container (the values describe the node, not the pod). `unknown` only when the file's origin cannot be classified.
- `type_id` values `1_102_001`ŌĆō`1_107_001` are provisional; they are internal constants and may change. What is fixed: field names/types/units/nullability, sort keys, semantics, scope.
- Units are stored raw (CPU ticks, meminfo KiB, PSI microseconds, kernel counters); the reader converts. Do not convert in the collector.

```rust
//! Reader for a `/proc` tree whose root is overridable for tests and for
//! host-mounted deployments.

use std::io;
use std::path::PathBuf;

/// A `/proc` root. Real collection uses `/proc`; tests and host-mounted pods
/// point it elsewhere via `KRONIKA_PROC_ROOT`.
#[derive(Debug, Clone)]
pub struct ProcFs {
    root: PathBuf,
}

impl ProcFs {
    /// Root from `KRONIKA_PROC_ROOT`, defaulting to `/proc`.
    #[must_use]
    pub fn from_env() -> Self {
        let root = std::env::var_os("KRONIKA_PROC_ROOT")
            .map_or_else(|| PathBuf::from("/proc"), PathBuf::from);
        Self { root }
    }

    /// A reader rooted at `root`.
    #[must_use]
    pub fn new(root: PathBuf) -> Self {
        Self { root }
    }

    /// Read `<root>/<rel>`, trimmed; empty content is an error.
    ///
    /// # Errors
    /// Returns the underlying `io::Error` (with the path) or an empty-file error.
    pub fn read(&self, rel: &str) -> io::Result<String> {
        let trimmed = self.read_raw(rel)?;
        let trimmed = trimmed.trim();
        if trimmed.is_empty() {
            return Err(io::Error::other(format!("{rel}: empty")));
        }
        Ok(trimmed.to_owned())
    }

    /// Read `<root>/<rel>` without trimming.
    ///
    /// # Errors
    /// Returns the underlying `io::Error`, tagged with the path.
    pub fn read_raw(&self, rel: &str) -> io::Result<String> {
        let path = self.root.join(rel);
        std::fs::read_to_string(&path)
            .map_err(|err| io::Error::new(err.kind(), format!("{rel}: {err}")))
    }
}
```

```rust
//! OS metric scope: what a value physically describes, so the reader never
//! mixes host-wide numbers with pod-local ones.

use crate::ProcFs;

/// Per-row scope tag. Stored as the `scope` `u8` column on every OS section.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum OsScope {
    Host,
    Pod,
    PodNet,
    Container,
    Unknown,
}

impl OsScope {
    /// Stable on-disk encoding.
    #[must_use]
    pub const fn as_u8(self) -> u8 {
        match self {
            Self::Host => 0,
            Self::Pod => 1,
            Self::PodNet => 2,
            Self::Container => 3,
            Self::Unknown => 4,
        }
    }
}

/// Whether `/proc/1/cgroup` content names a container runtime.
#[must_use]
pub fn detect_container_from_cgroup(cgroup: &str) -> bool {
    const MARKERS: [&str; 4] = ["kubepods", "docker", "containerd", "lxc"];
    cgroup.lines().any(|line| MARKERS.iter().any(|m| line.contains(m)))
}

/// Multi-signal container detection.
#[must_use]
pub fn detect_container(fs: &ProcFs) -> bool {
    if std::env::var_os("KUBERNETES_SERVICE_HOST").is_some() {
        return true;
    }
    if std::path::Path::new("/.dockerenv").exists() {
        return true;
    }
    fs.read_raw("1/cgroup")
        .is_ok_and(|c| detect_container_from_cgroup(&c))
}
```

```rust
//! Parse `/proc/stat` CPU lines (`1_102`) and the misc counters (`1_103`).

use std::num::ParseIntError;

/// One CPU's ticks; `cpu_id = -1` is the aggregate line.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub struct CpuRow {
    pub ts: i64,
    pub cpu_id: i32,
    pub user: i64,
    pub nice: i64,
    pub system: i64,
    pub idle: i64,
    pub iowait: i64,
    pub irq: i64,
    pub softirq: i64,
    pub steal: i64,
    pub guest: i64,
    pub guest_nice: i64,
}

/// Parse error for procfs lines.
#[derive(Debug)]
pub struct ParseError(pub String);

impl std::fmt::Display for ParseError {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        write!(f, "{}", self.0)
    }
}
impl std::error::Error for ParseError {}
impl From<ParseIntError> for ParseError {
    fn from(e: ParseIntError) -> Self {
        Self(format!("integer: {e}"))
    }
}

/// Parse every `cpu`/`cpuN` line. Ten time fields; missing trailing fields
/// (older kernels) default to `0` per the `/proc/stat` contract.
pub fn parse_cpu(content: &str, ts: i64) -> Result<Vec<CpuRow>, ParseError> {
    let mut rows = Vec::new();
    for line in content.lines() {
        let Some(rest) = line.strip_prefix("cpu") else {
            continue;
        };
        // Skip non-cpu lines that start with "cpu" prefix bytes but aren't cpu*.
        let (id_part, values) = match rest.split_once(char::is_whitespace) {
            Some((id, v)) => (id, v),
            None => continue,
        };
        let cpu_id = if id_part.is_empty() {
            -1
        } else {
            id_part
                .parse::<i32>()
                .map_err(|e| ParseError(format!("cpu id {id_part:?}: {e}")))?
        };
        let mut f = values.split_whitespace();
        let mut next = || -> Result<i64, ParseError> {
            f.next().map_or(Ok(0), |s| s.parse::<i64>().map_err(Into::into))
        };
        rows.push(CpuRow {
            ts,
            cpu_id,
            user: next()?,
            nice: next()?,
            system: next()?,
            idle: next()?,
            iowait: next()?,
            irq: next()?,
            softirq: next()?,
            steal: next()?,
            guest: next()?,
            guest_nice: next()?,
        });
    }
    if rows.is_empty() {
        return Err(ParseError("/proc/stat: no cpu lines".to_owned()));
    }
    Ok(rows)
}
```
